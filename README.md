# GoldEye（金眼）

一款轻量化的浏览器插件，实时显示东方财富网的黄金价格行情。

## 🎯 功能特性

- 📊 **实时金价展示** - 点击插件图标即可查看东方财富网实时黄金行情
- 🔄 **自动刷新** - 网页内容自动更新，无需手动操作
- 🎨 **简洁界面** - 360×500px 固定窗口，布局清晰美观
- ⚡ **极速加载** - 轻量设计，快速响应
- 🔒 **最小权限** - 仅使用必要的 `activeTab` 和 `tabs` 权限

## 📦 安装方法

### Chrome / Edge 浏览器

1. 下载或克隆本项目到本地
2. 打开浏览器扩展管理页面：
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `GoldEye` 项目文件夹
6. 插件安装成功！

## 🚀 使用说明

1. 点击浏览器工具栏中的 **GoldEye** 图标（金色眼睛标志）
2. 弹出窗口将显示东方财富网的实时黄金行情
3. 页面会自动刷新，保持数据最新

## 🔧 技术说明

- **Manifest 版本**: V3（符合最新 Chrome 扩展标准）
- **数据来源**: 东方财富网黄金行情页面
- **实现方式**: iframe 嵌入，无需额外 API 调用
- **兼容性**: Chrome、Edge 及其他 Chromium 内核浏览器

## ⚠️ 注意事项

- 本插件不采集、不存储任何用户数据
- 所展示的行情数据均来自东方财富网官方页面
- 插件不进行后台通信或数据爬取
- 如果页面禁止 iframe 嵌入，将自动在新标签页中打开
