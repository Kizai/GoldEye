# GoldEye（金眼）

一款轻量化的浏览器插件，实时显示东方财富网的黄金价格行情。

## 🎯 功能特性

- 📊 **实时金价展示** - 点击插件图标即可查看东方财富网实时黄金行情
- ⚡ **每秒更新** - 顶部实时行情条每 1 秒刷新一次（不依赖网页内脚本）
- 🔄 **自动刷新** - 下方网页内容定期刷新，并显示倒计时
- 🎨 **简洁界面** - 360×500px 固定窗口，布局清晰美观
- ⚡ **极速加载** - 轻量设计，快速响应
- 🔒 **最小权限** - 仅使用必要权限（`activeTab` / `tabs` / `storage` / `contextMenus` + 指定站点访问权限）

## 📦 安装方法

### Chrome / Edge 浏览器

1. 下载或克隆本项目到本地
2. 打开浏览器扩展管理页面：
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `GoldEye` 项目文件夹
6. 插件安装成功！

## 🚀 使用说明

1. 点击浏览器工具栏中的 **GoldEye** 图标（金色眼睛标志）
2. 弹出窗口将显示东方财富网的实时黄金行情
3. 页面会自动刷新，保持数据最新

## 🔧 技术说明

- **Manifest 版本**: V3（符合最新 Chrome 扩展标准）
- **数据来源**: 东方财富网黄金行情页面
- **实现方式**: iframe 展示网页 + 东方财富官方接口（`push2.eastmoney.com`）轮询，用于顶部每秒实时数字
- **兼容性**: Chrome、Edge 及其他 Chromium 内核浏览器

## ⚠️ 注意事项

- 本插件不采集、不存储任何用户数据
- 行情数据来自东方财富网页面与其公开接口
- 仅在弹窗打开期间发起请求用于更新顶部实时数字
- 如果页面禁止 iframe 嵌入，将自动在新标签页中打开
